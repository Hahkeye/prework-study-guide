HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta element is used to define items for the browser.
  • The section element is used to define a section in the document.
  • The main element is used to define the main content of the document.

Git

  • Git is a version control used to push and pull stuff from github or other cloud based services.
  • "git status" is used to check the current branch that is checked out.
  • "git checkout -b "branch-name"" creates a new branch under the name and switches to it.
  • "git commit -m "your message" used to commit changes.
  • "git merge [branch]" to merge specified branch into current.
